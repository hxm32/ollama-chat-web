Uses Ollama's hosted web search API as a tool the model can call. This requires internet access (separate from your local Ollama server) and an API key from your Ollama account. Sign in, then click Add API Key at ollama.com/settings/keys to generate one. Only models with tool-calling support (e.g. llama3.1+, qwen3, mistral-nemo) will actually use it.

⚠ Browsers block calling that API directly (CORS), it doesn't work without a proxy. Run the included ollama-search-proxy.py script (needs only Python, no install), then set the Proxy URL below to what it prints. Without a proxy, search will fail with a network error.
